About Parkinson's Disease

Parkinson's Disease (PD) is a progressive neurodegenerative disorder that affects movement. It occurs when nerve cells in the brain that produce dopamine become impaired or die. Early detection is crucial for better treatment outcomes.

Common Symptoms:
  • Tremors: Shaking in hands, arms, legs, jaw, or head
  • Muscle Rigidity: Stiffness and resistance to movement
  • Bradykinesia: Slowness of movement
  • Balance Issues: Impaired coordination and stability
  • Voice Changes: Speech and writing difficulties

1. Introduction

1.1 Background

Parkinson's Disease (PD) is a progressive neurodegenerative disorder affecting millions worldwide. Early detection is crucial for better treatment outcomes and improved quality of life. Traditional diagnostic methods rely on clinical observations and can be subjective, leading to delayed diagnosis.

1.2 Problem Statement

The current diagnostic process for PD is time-consuming and often requires multiple clinical visits. There's a need for a non-invasive, objective, and early detection method that can assist healthcare professionals in making accurate diagnoses.

1.3 Project Objectives
  • Develop an AI model for early PD detection using voice analysis
  • Achieve high accuracy and reliability in PD diagnosis
  • Create an interpretable model with clear feature importance
  • Provide a user-friendly interface for healthcare professionals

2. Literature Review

2.1 Voice Analysis in PD Detection

Voice analysis has emerged as a promising non-invasive method for PD detection. Research shows that voice characteristics change significantly in PD patients, even in early stages. These changes include:

  • Reduced vocal intensity
  • Increased voice tremor
  • Monotonic speech pattern
  • Reduced speech rate
2.2 Machine Learning Approaches

Various machine learning algorithms have been employed for PD detection:

  • Support Vector Machines (SVM)
  • Random Forest Classifiers
  • Neural Networks
  • K-Nearest Neighbors (KNN)
2.3 Feature Engineering Techniques

Key features used in voice analysis include:

  • Fundamental frequency measures
  • Jitter and shimmer analysis
  • Harmonic-to-noise ratio
  • Nonlinear dynamic measures
